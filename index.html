初中化学常见化学方程式完整版（修正版）

01、化合反应

4P + 5O₂ `点燃` 2P₂O₅ （红磷燃烧，产生白烟）
4P + 5O₂ = 2P₂O₅ （白磷自燃）
C + O₂ `点燃` CO₂ （木炭充分燃烧）
2C + O₂ `点燃` 2CO （木炭不充分燃烧）
S + O₂ `点燃` SO₂ （硫在氧气中燃烧）
3Fe + 2O₂ `点燃` Fe₃O₄ （铁丝在氧气中燃烧）
4Al + 3O₂ `点燃` 2Al₂O₃ （铝在氧气中燃烧/铝不易生锈的原因）
2Mg + O₂ `点燃` 2MgO （镁在空气中燃烧）
2Cu + O₂ `△` 2CuO （铜在空气中加热）
2H₂ + O₂ `点燃` 2H₂O （氢气燃烧）
C + CO₂ `高温` 2CO （二氧化碳通过炽热碳层）
CO₂ + H₂O = H₂CO₃ （二氧化碳溶于水）
CaO + H₂O = Ca(OH)₂ （生石灰制熟石灰）
2CO + O₂ `点燃` 2CO₂ （一氧化碳燃烧）
CaCO₃ + CO₂ + H₂O = Ca(HCO₃)₂ （石灰水变澄清）
H₂ + Cl₂ `点燃` 2HCl
2Na + Cl₂ `点燃` 2NaCl
3Mg + N₂ `点燃` Mg₃N₂
启示：燃烧不一定需要氧气参与。

02、分解反应

2HgO `△` 2Hg + O₂↑ （氧化汞分解）
2H₂O₂ `MnO₂` 2H₂O + O₂↑ （实验室制氧气）
2KMnO₄ `△` K₂MnO₄ + MnO₂ + O₂↑ （实验室制氧气）
2KClO₃ `MnO₂/△` 2KCl + 3O₂↑ （实验室制氧气）
过氧化氢分解法符合绿色化学理念，安全、环保。
2H₂O `通电` 2H₂↑ + O₂↑ （电解水）
CaCO₃ `高温` CaO + CO₂↑ （工业制生石灰）
2NaHCO₃ `△` Na₂CO₃ + H₂O + CO₂↑ （小苏打分解/灭火器原理）
Cu₂(OH)₂CO₃ `△` 2CuO + H₂O + CO₂↑ （碱式碳酸铜分解）
H₂CO₃ = H₂O + CO₂↑ （碳酸不稳定分解）
NH₄HCO₃ `△` NH₃↑ + CO₂↑ + H₂O （碳铵分解）
Ca(HCO₃)₂ `△` CaCO₃↓ + CO₂↑ + H₂O （钟乳石形成）

03、置换反应

H₂ + CuO `△` Cu + H₂O （氢气还原氧化铜）
C + 2CuO `高温` 2Cu + CO₂↑ （木炭还原氧化铜）
3C + 2Fe₂O₃ `高温` 4Fe + 3CO₂↑ （木炭还原氧化铁）
C + H₂O `高温` CO + H₂ （水煤气制取）
Zn + H₂SO₄ = ZnSO₄ + H₂↑ （实验室制氢气）
Fe + CuSO₄ = FeSO₄ + Cu （湿法炼铜）
Fe + 2AgNO₃ = Fe(NO₃)₂ + 2Ag （已修正）
Cu + 2AgNO₃ = Cu(NO₃)₂ + 2Ag
K、Ca、Na 太活泼，会先与水反应，不能置换盐中的金属。
2Mg + CO₂ `点燃` 2MgO + C
3Fe + 4H₂O(g) `高温` Fe₃O₄ + 4H₂

04、复分解反应

NaOH + HCl = NaCl + H₂O （中和反应）
Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O （胃舒平中和胃酸）
Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O （除铁锈）
CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑ （实验室制CO₂）
Na₂CO₃ + 2HCl = 2NaCl + H₂O + CO₂↑ （泡沫灭火器）
AgNO₃ + HCl = AgCl↓ + HNO₃ （检验Cl⁻）
BaCl₂ + H₂SO₄ = BaSO₄↓ + 2HCl （检验SO₄²⁻）
Ca(OH)₂ + Na₂CO₃ = 2NaOH + CaCO₃↓ （工业制烧碱）
NH₄Cl + NaOH `△` NaCl + NH₃↑ + H₂O （检验NH₄⁺）
CuSO₄ + 2NaOH = Cu(OH)₂↓ + Na₂SO₄ （生成蓝色沉淀）
FeCl₃ + 3NaOH = Fe(OH)₃↓ + 3NaCl （生成红褐色沉淀）
BaCl₂ + MgSO₄ = BaSO₄↓ + MgCl₂ （解毒原理）

05、其他重要反应

3CO + Fe₂O₃ `高温` 2Fe + 3CO₂ （高炉炼铁）
CH₄ + 2O₂ `点燃` CO₂ + 2H₂O （甲烷燃烧）
C₂H₅OH + 3O₂ `点燃` 2CO₂ + 3H₂O （乙醇燃烧）
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O （葡萄糖氧化）
CO₂ + Ca(OH)₂ = CaCO₃↓ + H₂O （石灰水变浑浊）
2NaOH + CO₂ = Na₂CO₃ + H₂O （NaOH变质原因）
2NaOH + SO₂ = Na₂SO₃ + H₂O
